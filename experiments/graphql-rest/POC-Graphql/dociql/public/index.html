GraphQL POC API Reference

Welcome to POC GraphQL documentation. Graphql is a query langage and and server-side runtime environment for application programming interfaces (APIs). It allows only the requested resources to be provided to clients. It is possible to deploy it within your own GraphiQL development environment.
